# AI Content Repurposing Studio

## Overview
Turn 1 piece of content into 10+ pieces automatically. Target: YouTube creators, podcasters, newsletters writers.

## Tech Stack
- **Frontend:** Next.js (Vercel deployment)
- **Payments:** Stripe
- **AI:** GLM-4.7 (already have via Clawdbot)
- **Storage:** Google Drive / Dropbox
- **Email:** SendGrid / Mailgun
- **Automation:** Custom Node.js scripts

## Directory Structure
```
content-studio/
├── landing-page/          # Next.js marketing site
├── intake/                # Order forms & client data
├── processing/            # AI content generation
├── delivery/              # Automated delivery system
├── outreach/              # Lead generation & email
├── scripts/               # Automation scripts
└── docs/                  # Documentation
```
